СГЕНЕРИРУЙ ОДНОСТРАНИЧНЫЙ САЙТ «MEADOW PARADISE RANCHO» НА РУССКОМ.

## 1) Общие требования

* Назначение: быстрое знакомство с ранчо, ответы на ключевые вопросы, удобная запись.
* Тон: спокойный, доверительный, без пафоса.
* Архитектура: one-page, якорное меню, липкий хедер.
* Технологии: React + Vite + TypeScript + TailwindCSS. Без внешних UI-китов. Анимации — Framer Motion по необходимости.
* Адаптив: мобильный первым. Ширина контейнера: max-w-[1200px]. Отступы секций: 80–120px desktop, 40–64px mobile.
* Доступность: контраст AA, focus-ring 2px, клавиатурная навигация.

## 2) Дизайн-система

* Палитра:

  * Поля пшеницы `#DCC7A1`
  * Хвойный зелёный `#2F5E4E`
  * Земляной коричневый `#7A5E3A`
  * Тёплый серо-бежевый `#EDE8E2`
  * Угольный текст `#1C1C1C`
  * Фон страницы — `#FFFCF7` или белый.
* Типографика:

  * Заголовки: Playfair Display или Cormorant Garamond.
  * Текст/UI: Inter или Montserrat.
  * H1: 44–56px, H2: 32–40px, H3: 22–24px, body: 16–18px, lead: 20px.
* Иконки: тонкий контур, без заливки.
* Эффекты: hover-scale 1.02, тень 8dp, duration-200ms; плавный якорный скролл 300–500ms.

## 4) Компоненты и поведение

* Header: прозрачный на Hero, после скролла — сплошной фон, тень, active-state по якорю.
* Buttons: первичная — хвойный зелёный, вторичная — контурная.
* Cards: скругление 16–20px, мягкая тень, изображение сверху, текст, CTA.
* Gallery: lazy-load, лайтбокс с свайпом на мобильных.
* FAQ: одиночное/множественное раскрытие, плавная высота.
* Form: in-place валидация, aria-описания для ошибок.
